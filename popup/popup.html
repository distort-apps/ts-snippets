Welcome to TS Snippets

TS Snippets is a Chrome extension that allows you to create, edit, and run TypeScript snippets directly in your browser's DevTools. With Monaco Editor support, you get IntelliSense, validation, and basic syntax colorization.

Features

  • Create new TypeScript snippets
  • Edit and save snippets
  • Run snippets with a click or using Command + Enter
  • Syntax highlighting and IntelliSense powered by Monaco Editor

Getting Started

To start using TS Snippets:

  1. Open Chrome DevTools (F12 or Ctrl+Shift+I)
  2. Go to the "TS Snippets" panel
  3. Create a new snippet and start coding!
